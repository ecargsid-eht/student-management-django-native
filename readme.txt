TO RUN DJANGO BACKEND:-

1. Go inside 'assignment_backend' folder and type command 'env\scripts\activate' using CMD
2. run 'py manage.py runserver' command.



TO RUN REACT NATIVE:-
1. Go inside 'assignment_front' folder and type command 'npm install'.
2. Type command 'npm start'.
3. Then type command 'npm run android'

IMPORTANT NOTE - This application of react native is build to run on an emulator or you can face some problems with api(axios).
